# Project file structure
```
coverly/
├── src # app itself /
│   ├── app # router /
│   │   ├── api # backend /
│   │   │   ├── auth /... # BetterAuth specific routes
│   │   │   └── spotify /... # Spotify data routes
│   │   ├── editor /... # editor route files
│   │   └── # root, layout and global css file
│   ├── components
│   │   ├── ui /
│   │   │   └── # shadcn/ui components
│   │   └── # custom components used in app
│   ├── lib /
│   │   └── # utility files
│   └── providers /
│       └── # react providers
├── public /
│   └── # assets
├── docs /
│   └── # project documentation
└── # other files
```
